## Adding News
1. create a new `.md` file in `content/blog/`
    * `cp content/blog/2017-08-seascapes.md content/blog/new-post.md`
2. edit file
    * set header data
    * add content in markdown format
    * any images should be uploaded to `static/img/blog/`
